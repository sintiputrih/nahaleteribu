That seems like quite a bit of bad luck but with the Gold cartridge I have a clue what might have happened if it was recent. One thing that might have happened was the battery used to keep the save portion of the cartridge alive has died. Luckily, the battery used is a 3 volt lithium button cell such as a CR2032, CR2025, or CR1616 for example. Those batteries can be found at just about any store except for the CR1616 as I think that is a special one. Unfortunately, you need to have electronics experience to do this as it does require soldering in order to replace the battery. Cheats and glitch abuse can result in the save file being corrupted which I think is what happened to your Blue version but I guess that can be solved by creating another save file. Since it is too late I assume for the Blue one this is just an FYI in case something like it happens again.
